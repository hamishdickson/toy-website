Part 1: The great focus debate...

So, first thing's first. I'm sorry for not updating you for a while, but I've been a busy man! Let me fill you in - I spend my days working for a financial software company in the city. Much of what I'm working on at the moment is producing the programs to make valuations and statements for the very very wealthy. I've been doing this a while and have found the tools we use good ... but not exactly what we need, so along with someone else we've been looking to write our own PDF generation tools. This is a massive piece of work ... and we are doing it in our free time ... which hasn't left much time for this.

Part 2: My frustration with Windows and desire to find something better

So far, this website has been written in Windows ... which sounds better than it is. You see, most of the hardcore geeks out there use a unix based operating system - Mac OS, Chrome or Linux - which means the best software is available there first (and it's maintained better there). That's causing me some problems and I'm going to have to move to another OS ... probably Ubuntu.mish, 22/12/2013
